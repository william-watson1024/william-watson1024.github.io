一、运动计划表（每周5天，循环训练）

训练日 训练类型 动作与安排 组数/时长 注意事项
周一 全身力量+有氧 1. 深蹲（标准或跳跃深蹲）
2. 俯卧撑（跪姿/标准）
3. 反向划船（用桌子边缘）
4. 平板支撑
5. 高抬腿跑（有氧）		 每个动作3组×15次
平板支撑3组×40秒
高抬腿3组×30秒		 动作速度放慢，感受肌肉发力
周二 核心+HIIT 1. 俄罗斯转体（徒手）
2. 仰卧交替抬腿
3. 侧平板支撑
4. HIIT循环：波比跳+登山跑+开合跳		 核心动作各3组×20次
HIIT 20秒运动+10秒休息，重复8轮		 HIIT后补充水分，心率保持中高强度
周三 休息/恢复 1. 散步或慢走
2. 全身拉伸（重点大腿后侧、肩背）		 30分钟低强度 避免久坐，每小时活动5分钟
周四 下肢+爆发力 1. 保加利亚分腿蹲（徒手）
2. 臀桥（单脚进阶）
3. 跳跃箭步蹲
4. 跳绳（无绳模拟）		 每个动作3组×12次/腿
跳绳3组×1分钟		 跳跃动作膝盖微屈，减轻关节压力
周五 上肢+代谢训练 1. 宽距俯卧撑（胸）
2. 钻石俯卧撑（肱三头肌）
3. 超人式（背部）
4. 折刀撑（肩）
5. 原地爬行（全身）		 每个动作3组×10次
原地爬行3组×8次		 背部训练时收紧肩胛骨
周六 全身功能性 循环训练：
1. 波比跳
2. 登山跑
3. 仰卧卷腹
4. 蛙跳
5. 动态平板		 每个动作45秒，休息15秒，循环3轮 根据体力调整强度，保证动作质量
周日 完全休息 冥想或泡沫轴放松（可用水瓶代替） 20分钟 重点放松小腿、下背部

二、饮食配合表（关键原则+示例）

目标 建议 一日三餐示例（参考1600-1800大卡）
减脂 热量缺口300-500大卡，碳水占40%，蛋白质30%，脂肪30% 早餐：2个鸡蛋+1片全麦面包+1杯菠菜
加餐：1小把坚果（10g）
增肌 蛋白质每公斤体重1.6-2g（如60kg需96-120g） 午餐：150g鸡胸肉+1碗糙米饭+西兰花
加餐：希腊酸奶+蓝莓
控糖 避免添加糖，主食选低GI（燕麦、红薯） 晚餐：120g煎三文鱼+藜麦沙拉+半根玉米
补水 每天2-3L水，运动时每15分钟喝100ml 运动后：香蕉1根+乳清蛋白粉（或1杯牛奶）

三、恢复策略表

恢复方式 具体执行 频率/时长
睡眠 保证7-9小时深度睡眠，睡前1小时避免蓝光 每天固定时间作息
拉伸 训练后做静态拉伸（如猫牛式、婴儿式、大腿前侧拉伸） 每次训练后10分钟
筋膜放松 用泡沫轴（或水瓶）滚动小腿、大腿外侧、上背部 每周3次，每次10分钟
主动恢复 休息日进行散步、瑜伽或游泳（低强度） 每周1-2次，30分钟
营养补充 运动后30分钟内补充蛋白质+快碳（如鸡蛋+香蕉） 每次训练后必需
